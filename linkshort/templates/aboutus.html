About App

This is a Django project used for url shrinking task. This project was made as the part of my project. This application allows users to input long URLs and receive a shorter link in return. This application simplifies sharing of web links, making it easier to distribute URLs across different platforms.

Project Info

This project was made by R Arjun Krishna as the part of Django project task.

Working

  • Enter the URL which should be shorten and Enter a title which will be the title for the data that is stored in database.
  • Click the Submit button.
  • The Shorten URL will be appeared.
  • Click the Reset button to shorten another URL.
  • For viewing saved urls, click on Dashboard on navigation bar.
  • Only 5 urls will be stored in Database. For deleting the url in database, use Delete button and Edit button can be used to edit the stored data.
  • The Search option is used to search for a particular url.
  • To logout, click on the Profile option on navigation bar.
